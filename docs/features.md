# 🚀 KeepSafe Features

KeepSafe is designed to provide a simple, secure, and efficient way to manage credentials across devices.

---

## 🔐 Security

Security is at the core of KeepSafe.

- Password hashing using **bcrypt**
- Secure communication via **HTTPS (TLS)**
- No sensitive data exposed in logs
- Clipboard auto-clean after copy (~30s)
- Neutral authentication responses (prevents user enumeration)

### 🔒 Future Enhancements
- End-to-End Encryption (E2E)
- Local encryption with AES-GCM
- Key derivation using Argon2id
- Recovery Code system

---

## 📱 Core Functionality

- Store credentials (login, password, notes)
- Fast search by title, login, or content
- Toggle password visibility
- One-tap copy to clipboard
- Optional login field per item

---

## ⭐ User Experience

- Clean and minimal interface
- Dark theme (Orbytek design system)
- Fast and responsive UI
- Designed for both mobile and desktop

---

## 🔔 Smart Notifications

- Password update reminders
- Configurable per item (X days)
- Works offline
- Fully compliant with Google Play policies
- No intrusive or unnecessary notifications

---

## 💾 Backup & Data Safety

- Local encrypted backup system (.keep)
- Manual export/import
- Works offline
- User-controlled security (no cloud dependency required)

---

## 🔄 Cross-Platform

- Android support
- Windows support
- Built with Flutter for consistent experience

---

## ⚙️ Technical Highlights

- Full-stack architecture
- PostgreSQL (Neon) backend
- Flutter frontend
- Modular and scalable codebase
- Production-ready deployment

---

## 🧠 Design Philosophy

KeepSafe follows three core principles:

- **Security first** → Your data belongs only to you  
- **Simplicity** → No unnecessary complexity  
- **Performance** → Fast, responsive, and reliable
